# NeuroPatron

Sistema de entrenamiento cognitivo y reconocimiento de patrones para estudiantes universitarios.

## Objetivo

NeuroPatron busca entrenar:

* razonamiento lógico
* reconocimiento de patrones
* automatización mental
* velocidad de resolución
* estructuras algorítmicas
* pensamiento computacional

La aplicación NO se enfoca en memorización tradicional.

Se enfoca en:

* aprendizaje adaptativo
* entrenamiento progresivo
* simulaciones reales
* ejercicios procedurales
* entrenamiento tipo reflejo
